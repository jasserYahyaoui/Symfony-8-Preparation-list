## Server side caching (Symfony HttpCache / reverse proxy) - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** When would you use Symfony HttpCache vs Varnish?

<details><summary>Click to reveal Solution</summary>

- **Symfony HttpCache**: Easy setup, built-in, no extra infrastructure. Good for small/medium apps.
- **Varnish**: Industrial-strength, C-based, much faster, supports ESI, VCL configuration. Use for high-traffic production environments.

</details>
