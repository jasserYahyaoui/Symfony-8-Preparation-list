## Components and Bridges - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** Without checking docs, classify each of these as Component, Bundle, or Bridge: `symfony/cache`, `symfony/monolog-bundle`, `symfony/twig-bridge`, `symfony/validator`, `symfony/security-bundle`.

**Hints:**
- Naming convention: `*-bundle` = Bundle, `*-bridge` = Bridge.
- Everything else is a Component.

**Testing:** `composer show <package-name>` to verify the type.

<details><summary>Click to reveal Solution</summary>

- `symfony/cache` → **Component**
- `symfony/monolog-bundle` → **Bundle**
- `symfony/twig-bridge` → **Bridge**
- `symfony/validator` → **Component**
- `symfony/security-bundle` → **Bundle**

</details>
