## Naming conventions - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** Fix these naming violations:
- Class: `user_service` →?
- Method: `GetUserById` →?
- Config key: `frameworkSecret` →?
- Route name: `appUserList` →?

<details><summary>Click to reveal Solution</summary>

- Class: `UserService`
- Method: `getUserById`
- Config key: `framework.secret`
- Route name: `app_user_list`

</details>
