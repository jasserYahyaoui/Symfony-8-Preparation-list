## Release management and roadmap schedule - Full Tutorial

> **Phase:** Phase 1 (For Initial Learning)


This is a theoretical topic. Quick reference:

```
Symfony Release Cycle:
- Minor release every 6 months (e.g., 8.0 → 8.1 → 8.2 → 8.3 → 8.4)
- LTS = last minor of a major (e.g., 7.4 is LTS for Symfony 7)
- Major release every 2 years
- Bug fixes: 8 months | Security fixes: 14 months (standard), 36 months (LTS)
```

No code exercise for this section — focus on memorizing the cycle for the exam.
