## Naming conventions - Full Tutorial

> **Phase:** Phase 1 (For Initial Learning)


Quick reference (no file needed):

```
Classes:       PascalCase  → MyCustomController, UserService
Methods:       camelCase   → getUserById(), handleRequest()
Variables:     camelCase   → $userName, $isActive
Constants:     UPPER_SNAKE → MAX_RETRIES, DEFAULT_LOCALE
Config keys:   snake_case  → framework.secret, security.firewalls
Service IDs:   snake_case  → app.service.user_manager (or FQCN)
Route names:   snake_case  → app_user_list, app_product_show
Template names: snake_case → user/show.html.twig
```
