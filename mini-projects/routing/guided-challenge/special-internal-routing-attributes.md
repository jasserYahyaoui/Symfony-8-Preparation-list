## Special internal routing attributes - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** What are the 4 special routing attributes that start with `_`? What does each do?

<details><summary>Click to reveal Solution</summary>

- `_controller`: Specifies which controller handles the route.
- `_format`: Sets the request format (json, xml, html) → affects content negotiation.
- `_fragment`: The URL fragment (`#section`).
- `_locale`: Sets the request locale for i18n.

</details>
