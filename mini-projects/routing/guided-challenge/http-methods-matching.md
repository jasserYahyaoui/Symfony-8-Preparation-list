## HTTP methods matching - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** What happens when a route matches the path but not the method? What HTTP status code is returned?

<details><summary>Click to reveal Solution</summary>

Symfony returns **405 Method Not Allowed** with an `Allow` header listing the accepted methods.

</details>
