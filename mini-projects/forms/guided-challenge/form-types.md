## Form types - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** What is the difference between `expanded=true/false` and `multiple=true/false` on a `ChoiceType`?

<details><summary>Click to reveal Solution</summary>

| expanded | multiple | Rendered as |
|----------|----------|------------|
| false | false | `<select>` dropdown (single) |
| false | true | `<select multiple>` |
| true | false | Radio buttons |
| true | true | Checkboxes |

</details>
