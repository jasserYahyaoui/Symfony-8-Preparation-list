## Form themes - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** What are the 3 built-in form themes Symfony provides?

<details><summary>Click to reveal Solution</summary>

1. `form_div_layout.html.twig` — Default (each row in a `<div>`).
2. `form_table_layout.html.twig` — Renders in a `<table>`.
3. `bootstrap_5_layout.html.twig` — Bootstrap 5 styled.
4. `tailwind_2_layout.html.twig` — Tailwind CSS styled.

</details>
