## Console component and FrameworkBundle - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** What are the 4 lifecycle methods of a Symfony command?

<details><summary>Click to reveal Solution</summary>

1. **`configure()`**: Define name, description, arguments, options.
2. **`initialize(InputInterface, OutputInterface)`**: Pre-execution setup.
3. **`interact(InputInterface, OutputInterface)`**: Prompt user for missing arguments interactively.
4. **`execute(InputInterface, OutputInterface)`**: The main command logic. Returns status code (0 = success).

</details>
