## Console events - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** What are the 3 console events and when does each fire?

<details><summary>Click to reveal Solution</summary>

1. **`ConsoleEvents::COMMAND`**: Fires BEFORE a command runs. Can prevent execution.
2. **`ConsoleEvents::TERMINATE`**: Fires AFTER a command finishes. Receives exit code.
3. **`ConsoleEvents::ERROR`**: Fires when a command throws an exception.

</details>
