## Profile and profiler usage in tests - Guided Challenge

> **Phase:** Phase 2 (For Revision)


**Goal:** What collectors are available in the Profiler, and what can you check with each?

<details><summary>Click to reveal Solution</summary>

| Collector | Purpose |
|-----------|---------|
| `time` | Request duration |
| `memory` | Memory usage |
| `exception` | Check for exceptions |
| `request` | Status code, headers, route |
| `router` | Matched route info |
| `events` | Dispatched events |
| `logger` | Log entries |
| `db` (Doctrine) | SQL queries |
| `mailer` | Sent emails |
| `validator` | Validation errors |

</details>
