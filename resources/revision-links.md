# Ressources de révision recommandées

## Officiel
- FAQ certification: https://certification.symfony.com/faq.html
- Détails examen Symfony: https://certification.symfony.com/exams/symfony.html
- Documentation Symfony 8.0: https://symfony.com/doc/8.0/

## Retours d’expérience
- IFD Attic (experience report): http://www.ifdattic.com/my-experience-sensiolabs-symfony-certification-exam/
- Abetari (prise de notes): https://www.abetari.com/symfony-certification-prise-de-notes/

## Communautaire
- Certificationy (QCM/ressources): https://github.com/certificationy/certificationy

## Conseils d’usage
- Prioriser d’abord les ressources officielles.
- Utiliser les retours d’expérience pour ajuster la stratégie (temps, stress, formulation).
- Utiliser les ressources communautaires en entraînement complémentaire, pas en source de vérité.
